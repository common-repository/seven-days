=== Seven Days ===
Contributors: Apichai, ratakorn, zeally
Tags: widget, day, condition, ajax, weekly
Requires at least: 2.8
Tested up to: 3.2.1
Stable tag: 0.2.2

Automatically show/hide your widget daily with "Seven Days", let you set widget to show / hide by day condition  (Sun - Sat)

== Description ==

* set widget to show / hide by day condition  (Sun - Sat)

= Feature: =
* Show/Hide on specific day for "Widget"
* Very friendly UI

= Use cases =
* Daily food menu : restaurant
* Daily notice : shop
* Daily schedule : school

== Installation ==

1. Download Seven Days.
2. Extract & upload “seven-days” folder to /wp-content/plugins/ directory.
3. Go to WordPress admin – Plugins page and activate “seven-days” plugin
